# Snake Game

A classic Snake game built with JavaScript that includes both manual play and an AI mode.

[Software Demo Video](https://www.loom.com/share/90e52b8585d14ed4b2c266571fc6a0f8)

## Features

- Classic Snake gameplay
- AI mode that plays automatically
- Multiple difficulty levels (Easy, Medium, Hard)
- Score tracking with high score saved locally
- Pause/Resume functionality
- Clean and simple interface

## How to Play

1. Open `index.html` in a web browser
2. Click "Start" to begin the game
3. Use arrow keys to control the snake
4. Eat the red food to grow and increase your score
5. Avoid hitting the walls or your own body
6. Press Space to pause/resume
7. Click "AI Mode" to watch the AI play

## Controls

- **Arrow Keys**: Move the snake
- **Space**: Pause/Resume
- **Start Button**: Begin a new game
- **Pause Button**: Pause or resume the game
- **Restart Button**: Reset the game
- **AI Mode Button**: Toggle AI control

## Difficulty Levels

- **Easy**: Slower game speed (200ms)
- **Medium**: Normal game speed (150ms)
- **Hard**: Faster game speed (100ms)

## Files

- `index.html` - Main HTML file
- `css/style.css` - Styling
- `js/game.js` - Main game logic
- `js/snake.js` - Snake class
- `js/ai.js` - AI algorithm

## Requirements

Just a modern web browser. No installation needed.
