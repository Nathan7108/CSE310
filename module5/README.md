# Drawing App

A simple drawing and painting web app built with TypeScript.

[Software Demo Video](https://www.loom.com/share/your-video-link-here)

## Features

- Draw on canvas with mouse or touch
- Color picker for choosing colors
- Adjustable brush size
- Clear canvas
- Undo functionality
- Save drawings as PNG images
- Works on desktop and mobile

## How to Use

1. Install dependencies: `npm install`
2. Compile TypeScript: `npm run build`
3. Open `index.html` in a web browser
4. Select a color and brush size
5. Draw on the canvas
6. Use Clear to erase everything
7. Use Undo to remove last action
8. Use Save to download your drawing

## Controls

- **Color Picker**: Choose drawing color
- **Brush Size Slider**: Adjust brush thickness (1-50)
- **Clear Button**: Erase entire canvas
- **Undo Button**: Remove last drawing action
- **Save Button**: Download drawing as PNG

## Files

- `index.html` - Main HTML file
- `css/style.css` - Styling
- `src/app.ts` - Main TypeScript code
- `tsconfig.json` - TypeScript configuration
- `package.json` - Dependencies

## Requirements

- Node.js and npm
- Modern web browser
- TypeScript compiler

## Development

```bash
npm install
npm run build
```

For development with auto-compilation:
```bash
npm run watch
```
